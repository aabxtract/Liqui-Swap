# **App Name**: LiquiSwap

## Core Features:

- Token Swapping: Allows users to swap between ERC-20 tokens using the x*y=k formula, incorporating a 0.3% fee per swap.
- Liquidity Provision: Enables users to add liquidity to token pairs, receiving LP tokens in proportion to their contribution.
- Liquidity Removal: Allows users to remove their liquidity, receiving the corresponding amount of each token from the pair.
- Automated Pricing: Automatically determines token prices using the constant product formula (x*y=k), ensuring balanced liquidity pools.
- Slippage Protection: Protects users from excessive slippage during swaps by allowing them to set slippage tolerance parameters.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a trustworthy, serious, and modern feel.
- Background color: Light lavender (#E6E6FA) providing a soft contrast to the primary color.
- Accent color: Electric violet (#8F00FF) for interactive elements and highlights, analogous to indigo and offering vibrant contrast.
- Font: 'Poppins', a grotesque-style sans-serif, providing a modern, objective, neutral look for both headlines and body text. Alternate: Montserrat.
- Minimalist icons representing token pairs and transaction types.
- Clean and intuitive layout with clear sections for swapping, providing liquidity, and viewing balances.
- Smooth transitions and subtle animations on interactions, like token swaps and liquidity changes.
